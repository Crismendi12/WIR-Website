# WIR Innovation - AI Platform

Website oficial de WIR Innovation con dashboards inteligentes y automatización de flujo de trabajo.

## 🚀 Características

- **Dashboard en Tiempo Real**: Visualización de datos con SSA Sales Intelligence y UCP Underwriter Intelligence
- **Flujo Automatizado**: Sistema de procesamiento de submissiones de 6 etapas
- **Propuesta de Valor Única**: Asociaciones estratégicas con MAHWAY, AVANTE y WIR
- **Diseño Moderno**: Interfaz construida con React, Tailwind CSS v4 y Motion

## 🛠️ Tecnologías

- **React** 18.3.1
- **Vite** 6.3.5
- **Tailwind CSS** 4.1.12
- **Motion** (Framer Motion) 12.23.24
- **React Router** 7.13.0
- **Lucide Icons**
- **Recharts** para visualizaciones

## 📦 Instalación

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev

# Build para producción
pnpm build
```
